# MessengerProject ✉️

Сучасний вебдодаток-месенджер на Flask для швидкого обміну повідомленнями між користувачами в реальному часі.

## 🛠️ Функціонал додатка
* **Обмін повідомленнями:** Можливість надсилати повідомлення та переглядати історію чатів.
* **Авторизація за нікнеймом:** Реєстрація та вхід користувачів без прив'язки до електронної пошти.
* **Збереження історії:** Робота з базою даних для надійного збереження профілів користувачів та надісланих повідомлень.
* **Безпека:** Захист сесій користувачів, хешування паролів та обов'язкова валідація форм через CSRF-токени.

## 🚀 Стек технологій
* Python (Flask, Flask-Login, Flask-WTF)
* База даних (SQLite3 з використанням row_factory для оптимізації запитів)
* HTML5, CSS3 (стильний та лаконічний інтерфейс чату)
